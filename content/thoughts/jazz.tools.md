[Jazz](https://jazz.tools/) is an *open-source* multiplayer data toolkit. Build *live*, *collaborative*, *multi-device* apps *with frontend code only.*

Embodies the principles of [[local-first software|local-first software](local-first%20software.md)

Components

1. `cojson`: A collaborative JSON type
2. State synchronization and persistence
3. Cryptographic [[access control|access control](access%20control.md)
4. Global mesh: smart routing transport layer for p2p data. With [[encryption.md|encryption]]!
