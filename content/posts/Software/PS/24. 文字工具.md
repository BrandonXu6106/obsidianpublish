## 1. 快捷键

| 快捷键            | 功能        |
| -------------- | --------- |
| `numpadenter`  | 结束输入      |
| `enter`        | 换行        |
| `alt → `       | 增加**字间距** |
| `alt →`        | 减小**字间距** |
| `ctrl alt ↓`   | 增大**行间距** |
| `ctrl alt ↓`   | 缩小**行间距** |
| `ctrl shift >` | 增大**字号**  |
| `ctrl shift <` | 缩小**字号**  |

### 2.2. 路径文字
- **绘制方法**：
	- `钢笔工具`：画出路径
	- `文字工具`：鼠标移动到路径上，点一下路径
	  ![202604221523900.gif|390](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604221523900.gif)
- **调整文字位置方向**：
	- `小白工具`：到路径上点住不动，拖动
	  ![202604221527415.gif|413](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604221527415.gif)


### 2.3. 内部文字
- **绘制方法**：
	- `钢笔工具`：画出形状路径
	- `文字工具`：鼠标移动到路径内部，光标上出现圆圈，点一下
	  ![202604221602537.gif|410](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604221602537.gif)


### 2.4. 字距微调
- 功能图标：
  ![222](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604221610026.png)
- 方法：光标定位到两个字符之间，`alt ←` 缩短字符间距
