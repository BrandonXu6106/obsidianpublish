## 1. 修复画笔工具和图案图章工具的区别
- 修复画笔：复制并融合
- 图案图章：复制并覆盖 ^Fh18GXHX
## 2. 渐变
- 仿色：默认勾选，差别不大
- 渐变滑块：上方滑块调整**透明度**；下方滑块调整颜色
- 方法：默认可感知，变化不大
## 3. 渐变案例（金属圆形）
- 渐变背景：
	- 新建图层
	- 深灰色→浅灰色
	- 滤镜-杂色-添加杂色-勾选单色-2% ^oN0wPZde
- 外圆
	- 中心定位
		- Ctrl+a 全选-ctrl+t-ctrl+r 调出标尺-上下拖两条参考线至中心 - 回车 - ctrl+d 去掉选取
	- 中心圆形
		- 圆形选区工具-从中心向外 shift+alt 拉出一个圆
	- 颜色
		- 新建图层 - 渐变颜色 - 四个灰色四个浅灰色相间隔 - 最末端复制一个和起始色块一样的颜色色块（首位颜色相连）- 调整色块间隔（首位间距加起来=中间任意两个之间的间距）- 模式选扇形渐变 - 从中心向外拉出渐变 - ctrl+d 取消选区 - ctrl+t 旋转调整角度
			-  tips：先点击想要复制的色块，再在空白区点一下，就可以复制刚刚点的色块颜色
	- 边缘厚度
		- 边缘圆环
			- 复制一层 - ctrl+左击缩略图，载入选区 - 点击选区工具 - 右击 - 变换选区 - alt+shift 缩小选区 - 回车 - del 删除 - ctrl+t 后稍微旋转（内外颜色错开）- 添加一点杂色
		-  圆环阴影
			- 复制一层圆环 - ctrl+t 后 shift+alt 稍微缩小一半 - 回车 - ctrl+U 降低明度 - 将他置于第一个圆环下方 - 高斯模糊 - ctrl+shift+; 取消参考线
		- 合并两图层
	- 线性纹理
		- 在圆形图层之上复制一个图层 - 滤镜 - 杂色 - 添加在杂色 15%左右（稍微大一点）- 载入选区- 滤镜 - 模糊 -径向模糊 - 旋转 - 确定 - ctrl+d 取消选区 - 模式选择叠加模式 - 拉低不透明度（降低点亮度）
		- 合并两图层
- 内圆
	- 形状
		- 外圆上方复制一个外圆 - ctrl+T 后中心缩小 - 载入选区 - 隐藏该图层
		- 选中底层外圆 - del 删除
		- 打开隐藏图层 - ctrl+t - 中心缩小 - 回车
	- 边框
		- 在上方复制一层内圆 - 中心缩小一点 - ctrl+t 旋转（做出颜色层次）
		- 选中下层内圆 - ctrl+u 拉低明度
- 镂空圆
	- 镂空图层
		- Ctrl +shift +; 调出辅助线 - 从中心拉出一个比内圆稍大点的圆形选区并上移该选区 - 新建图层 - 填充黑色
	- 环形复制
		- Ctrl+t - 按住 alt+shift 键将中心点垂直拖动到内圆正中心 - 选项栏将角度改为 60°（360/6） - 敲两下回车 - 按住 crl+shift+alt 后按 T 键，按五次
	- 镂空
		- 合并六个图层 - 载入选区 - 删除镂空图层 - 选中外圆图层 - del 删除 - 隐藏辅助线
- 金属厚度
	- 选中内圆 - 混合选项 - 斜面和浮雕 - 复位为默认值 - 取消勾选使用全局光 - 阴影处拖动角度 117 - 结构中大小拉大至五个像素 - 高光不透明度加大些 - 阴影不透明度降低些
- 影子
	- 新疆图层 - 柔边缘画笔 - 100%不透明 - 稍微拉大画笔 - 点一个黑点 - ctrl+t 拉扁一点 + alt 键两方拉长 - 放置底部

## 4. 减淡加深海绵工具
- 减淡工具：增加亮度，降低饱和度
- 加深工具：降低亮度，加强饱和度
- 海绵工具：去色/加色（局部）
## 5. 钢笔工具
- 让路径隐藏
	- 按两次 esc 键（第一次取消选中的路径，第二次隐藏路径）
- 选择单个锚点
	- 按住 ctrl 点击锚点=小白工具
- 脱离上一个锚点，绘制新路径
	- 按住 ctrl 在空白处点一下
