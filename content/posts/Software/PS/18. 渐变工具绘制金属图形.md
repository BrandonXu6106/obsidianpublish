# 终稿
![202604070248637.png|371](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604070248637.png)

## 1. 背景色(渐变)：
- `新建图层`
- `渐变预设`：深灰色→浅灰色
- **渐变方向**：从上到下
- **杂色质感**：`滤镜` -- `杂色` -- `添加杂色` -- `勾选单色` -- `数量：2%` ^oN0wPZde

 
## 2. 外圆
### 2.1. 中心定位
- `Ctrl+a` 全选 -- `ctrl+t`效果控件--`ctrl+r` 调出标尺
- 拖出两条参考线至中心后，`回车`
  ![256](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171752793.png)
- `ctrl+d` 去掉选取

### 2.2. 中心正圆
- `新建图层`
- `圆形选区`：`shift+alt` 从中心向外，拉出一个圆
  ![271](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171805462.png)
### 2.3. 金属颜色
- **渐变预设**：
	- **颜色**：四个==中灰色==，四个==中白色==，==间隔==相排 
		- TIPS：如何==首尾颜色相连==，使得过渡顺滑
			1. 首尾颜色相同
			2. 调整色块==间隔==：缩短首端两个滑块间的距离，缩短尾端两个滑块间的距离，其他大致均分
			   ![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171832737.png)
	- **类型**：`角度渐变 `
		- 从中心向外拉出渐变
		  ![202604171832728.png|290](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171832728.png)

### 2.4. **调整方向**
- `ctrl+d` 取消选区
- `ctrl+t` 旋转，调整角度，`回车`
  ![202604171835562.png|270](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171835562.png)

### 2.5. 边缘厚度
- **边缘圆环**
	- 复制：复制金属圆图层，隐藏原来的图层
	- 创建选区：`ctrl+左击`缩略图，载入圆形选区 
	- 缩小选区：`选区工具` 
		- 右击图层 -- `变换选区` -- `alt+shift` 适当缩小选区 -- `回车 `
		  ![202604171847198.png|268](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171847198.png)
	- 删除中心区域：`del`，再
	   ![202604171848496.png|331](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171848496.png)
	- 调整角度：
		- `ctrl d`取消选区
		- `ctrl+t` 稍微旋转（使得内外颜色错开）
		- 显示原先的金属图层观察
		  ![276](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171853695.png)
- **圆环阴影**
	- 复制一份圆环图层
	- 缩小图层：`ctrl+t` 后 `alt` 稍微缩小一半 -- `回车`
	- 调整颜色：`ctrl+U` 降低明度 
	- 调整位置：将他置于第一个圆环的下方 
	  ![202604171859323.png|392](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171859323.png)
	- 高斯模糊：滤镜 -- 模糊层-- 高斯模糊
	  ![202604171907276.png|384](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171907276.png)
	- 隐藏参考线：`ctrl+;` 
- **合并图层**(可选)
### 2.6. 特殊纹理
- **制作纹理层**
	- 复制一层金属圆图层
	- 添加杂色效果：`滤镜` -- `杂色` -- `添加杂色`-- `单色` 
	  ![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171926534.png)
	- 添加线性纹理：载入选区 -- `滤镜` -- `模糊` --`径向模糊` -- `旋转` -- 确定 -- `ctrl+d` 取消选区 
	  ![202604171930982.png|322](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171930982.png)
- **模式选择**：`叠加模式 `
- **降低点亮度**：拉低不透明度
  ![202604171937788.png|348](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604171937788.png)
- 合并两图层 (可选)


## 3. 内圆
- **形状**
	- 复制一个金属圆图层
	- `ctrl+T`调出效果控件，按 `alt` 中心缩小至适合位置
	- `ctrl 左键` 载入选区，并隐藏该图层
	  ![202604181818667.png|330](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604181818667.png)
	- 选中原来的金属圆图层, `del` 删除
	  ![202604181823176.png|327](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604181823176.png)
	- 打开隐藏的图层 -- `ctrl+t` -- 中心缩小 -- 旋转角度 -- 回车
	  ![202604182259111.png|324](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604182259111.png)
- **外轮廓**
	- `ctrl j` 复制一层内圆 -- 中心缩小一点
	- `ctrl+t` 旋转（做出颜色层次）
	  ![355](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604182304607.png)
	- 选中下层内圆 -- `ctrl+u` 拉低明度


## 4. 镂空圆
- **镂空圆**
	- `Ctrl +shift +`：调出辅助线 
	- `选区工具`：从中心拉出一个==比内圆稍大点==的圆形选区，并`shift`上移该选区
	- `新建图层` -- 填充黑色
	  ![202604182329675.png|315](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604182329675.png)
- **环形圆复制**
	- `Ctrl+t` -- 按住 `alt+shift` 键将中心点垂直拖动到内圆正中心
	  ![202604182344287.gif|337](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604182344287.gif)
	- 在选项栏，将角度改为 ==60°（360/6）== -- 敲两下回车
	  ![202604182346084.png|381](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604182346084.png)
	- 按住 `crl+shift+alt`，连续按==五次==`T`键
	  ![384](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604182350557.gif)
- **镂空金属圆**
	- `ctrl e`：合并六个环形圆图层
	- 载入选区，删除/隐藏环形圆(环形圆已经无用)
	  ![202604190001567.png|377](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604190001567.png)
	- 选中金属圆图层，`del` 删除
	  ![202604190005880.png|369](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604190005880.png)
	- `ctrl ;` 隐藏辅助线，`ctrl d` 取消选区
- **镂空圆厚度**
	- 选中金属圆图层
	- `fx` -- `混合选项`
		- `斜面和浮雕`
			- 点击下方的`复位为默认值`，并取消勾选`使用全局光` 
			- `阴影`
				- 角度：117 
				  ![241](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202604190020744.gif)
				- 高光模式：不透明度加大至84
				- 阴影模式：不透明度降低至3
			- `结构`
				- 大小：五个像素
				- 软化：1个像素
