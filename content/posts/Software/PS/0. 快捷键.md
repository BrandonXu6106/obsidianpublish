## 1. 选区

| 快捷键                          | 功能       |
| ---------------------------- | -------- |
| `Ctrl 回车`                    | 路径变选区    |
| `Ctrl shift I` -- **I**nvert | 反选选区     |
| `Ctrl J` -- **J**ump         | 选区复制为新图层 |

| 快捷键                      | 功能     |
| ------------------------ | ------ |
| `ctrl a`                 | 建立画框选区 |
| `Ctrl D` -- **D**eselect | 取消选区   |
| `Ctrl`                   | 选择     |
| `Shift`                  | 添加     |
| `Alt`                    | 减去     |
| `Shift alt`              | 交集     |

| 快捷键                    | 功能     |
| ---------------------- | ------ |
| `Ctrl 左键缩略图`           | 建立图层选区 |
| `Ctrl shift 左键缩略图`     | 添加图层选区 |
| `Ctrl alt 左键缩略图`       | 减去图层选区 |
| `Ctrl shift alt 左键缩略图` | 交叉图层选区 |

| 快捷键                 | 功能    |
| ------------------- | ----- |
| `Ctrl 回车`           | 新建选区  |
| `Ctrl shift 回车`     | 添加到选区 |
| `Ctrl alt 回车`       | 减去选区  |
| `Ctrl shift alt 回车` | 交叉选区  |
|                     |       |

## 6. 工具
### 6.1. 蒙版

| 快捷键               | 功能            |
| ----------------- | ------------- |
| `shift 左击(蒙版缩略图)` | 取消/浏览蒙版效果     |
| `alt`  点击(新建蒙版按钮) | 建立蒙版，==隐藏区域== |

### 6.2. 画笔

| 快捷键                 | 功能   |
| ------------------- | ---- |
| `]` / `alt` 右键 向右拖拽 | 放大笔触 |
| `[` / `alt` 右键 向左拖拽 | 缩小笔触 |
| `alt` 右键 向下拖拽       | 增加硬度 |
| `alt` 右键 向上拖拽       | 减小硬度 |

### 6.3. 钢笔工具

| 快捷键            | 功能            |
| -------------- | ------------- |
| 按住`ctrl`       | 变成小白工具        |
| `ctrl` 左击空白处   | 脱离上一个锚点，绘制新路径 |
| `alt`          | 转换点工具         |


### 6.4. 字体工具

| 快捷键            | 功能        |
| -------------- | --------- |
| `numpadenter`  | 结束输入      |
| `enter`        | 换行        |
| `alt → `       | 增加==字间距== |
| `alt →`        | 减小==字间距== |
| `ctrl alt ↓`   | 增大==行间距== |
| `ctrl alt ↓`   | 缩小==行间距== |
| `ctrl shift >` | 增大==字号==  |
| `ctrl shift <` | 缩小==字号==  |


### 6.5. 变换工具

| 快捷键                 | 功能              |
| ------------------- | --------------- |
| `ctrl shift alt t ` | 执行上次的命令，并==复制== |
| `ctrl shift t `     | 执行上次的命令，==不复制== |
| `按住 ctrl alt shift` | 透视              |
