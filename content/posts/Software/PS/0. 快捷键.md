## 1. 选区

| 快捷键            | 功能       |
| -------------- | -------- |
| `Ctrl 回车`      | 路径变选区    |
| `Ctrl shift I` | 反选选区     |
| `Ctrl J`       | 选区复制为新图层 |

| 快捷键         | 功能     |
| ----------- | ------ |
| `ctrl a`    | 建立画框选区 |
| `Ctrl D`    | 取消选区   |
| `Ctrl`      | 选择     |
| `Shift`     | 添加     |
| `Alt`       | 减去     |
| `Shift alt` | 交集     |

| 快捷键                    | 功能     |
| ---------------------- | ------ |
| `Ctrl 左键缩略图`           | 建立图层选区 |
| `Ctrl shift 左键缩略图`     | 添加图层选区 |
| `Ctrl alt 左键缩略图`       | 减去图层选区 |
| `Ctrl shift alt 左键缩略图` | 交叉图层选区 |

| 快捷键                 | 功能    |
| ------------------- | ----- |
| `Ctrl 回车`           | 新建选区  |
| `Ctrl shift 回车`     | 添加到选区 |
| `Ctrl alt 回车`       | 减去选区  |
| `Ctrl shift alt 回车` | 交叉选区  |
|                     |       |

## 2. 系统

| 快捷键                | 功能     |
| ------------------ | ------ |
| `Ctrl shift alt K` | 快捷键设置  |
| `Ctrl K`           | 首选项    |
| `Ctrl R`           | 标尺     |
| `Ctrl ；`           | 隐藏参考线  |
| `Ctrl J`           | 复制到新图层 |
|                    |        |

| 快捷键            | 功能   |
| -------------- | ---- |
| `Ctrl N`       | 新建项目 |
| `Ctrl shift N` | 新建图层 |

## 3. 图层

| 快捷键            | 功能  |
| -------------- | --- |
| `Ctrl [`       | 下移  |
| `Ctrl ]`       | 上移  |
| `Ctrl shift [` | 置底  |
| `Ctrl shift ]` | 置顶  |


| 快捷键                    | 功能   |
| ---------------------- | ---- |
| `Ctrl E`               | 合并图层 |
| `Ctrl  shift  alt + E` | 盖印图层 |

## 4. 颜色

| 快捷键          | 功能      |
| ------------ | ------- |
| `Alt del`    | 填充前景色   |
| `Ctrl del`   | 填充背景色   |
| `Shift back` | 弹出填充对话框 |
| `Del`        | 填充透明    |
| `D`          | 切换为黑白背景 |

| 快捷键            | 功能    |
| -------------- | ----- |
| `Ctrl I`       | 反相    |
| `Ctrl shift U` | 去色    |
| `Ctrl L`       | 色阶    |
| `Ctrl M`       | 曲线    |
| `Ctrl B`       | 色彩平衡  |
| `Ctrl U`       | 色彩饱和度 |

## 5. 数值调整
| 快捷键                               | 功能     |
| --------------------------------- | ------ |
| 选中图层，快速敲数字键盘                      | 调整透明度  |
