## 文本设置
### 文本属性
- `color`：文本**颜色**，使用颜色名或 `#212131` 这样的编码
- `line-height`：每行文本的**高度**，用**像素单位**设置
- `text-align`：文本的**对齐**属性
	- 包含 **left、right、center** (居中)
- `text-decoration`：文本的**修饰内容**
	- 包含 **none、underline（下划线）、line-through（删除线）** 等设置
- `word-spacing`：**字间距**，用**像素单位**设置


### 字体属性
- `font-family`：设置文字的**字体**
- `font-size`：文字的**字号**，用**像素**表示
- `font-style`：字体**风格**
	- 包含 **normal**（默认）、**italic**（斜体）、**oblique** (倾斜)
- `font-weight`：字体的**字重**，可以用**数字**也可以字体**自带字重**


浏览器对文字的**渲染**是**通过用户本地的字库**来完成的，所以字体**只用常见字体**完成

### 列表属性:
- `list-style-image`：列表项**符号替换**成**图片**的设置;![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602162015164.png)
- `list-style-position`：列表项符号**图片所处**的位置;
- `list-style-type`：列表项**符号的类型**
	- 包含none、disc（实心圆）、circle（空心圆）suqare（方形)、decimal（数字）等;


## 背景设置
- background的作用
	- `background-color`：定义**背景色**的属性
	- `background-image`：定义**背景图片**的**属性**，使用"url （img/logo.png )" 的方法
	- `background-position`：定义**背景图片**的x、y轴距离标签左上角的**距离**
	- `backgorund-repeat`：定义图片**重复的规则**(平铺设置)
		- 在网页中，无法右键保存的图片


## 排版定位属性
>一个完整的网页，是由**无数矩形拼接**起来的一一**矩阵视图**

在完成页面的排版中，需要应用相关的**定位属性**：
- `position`：定位类型设置
	- static(**默认位置**，没有定位)
	- relative(相对定位）-- **相对**于**默认位置**进行定位
		- 因此，"left:20" 会向元素的 LEFT 位置添加 20 像素。
	- absoulte(绝对定位) -- **相对**于static定位以外的**第一个父元素**进行定位。
		- 元素的位置通过“left","top","right" 以及“bottom"属性进行规定。
	- fixed(固定定位) -- **相对**于**浏览器窗口**进行定位。 
		- 元素的位置通过“left"，"top"，“right”以及“bottom”属性进行规定。
- `z-index`： 页面的 z 轴层级设置，用数字表示序列。
	- 约等于设计软件中的**图层顺序**通过**序号**来进行**控制**==乙轴上下排序==![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602170032409.png)
- `float`：元素的**浮动定位**设置，**位置不固定**，排列可以发生**响应式变化**![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602170038673.png)
	- 专门用来创建**一行多列**的复用样式(固定宽)，并可以让标签自动产生block的属性
	- 可以设置left、right
- `clear`：**清除浮动**元素的**属性**，包含 left、right、both、none。
- `top/left/right/bottom`： 上左右下方向的**偏移距离**设置。

定位类型是一个比较复杂的概念，相当于我们设置标签的”中心点“要关联哪个元素![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602162246972.png)
## 动画相关设置
### 过渡型
`transition`是一个专门用来==控制动画过度==的属性，它包含：
- `transition-property`：需要过渡的 **CSS 名称**，填写属性名
- `transition-duration`： 动画**执行**的**时长**，填写**秒数**即可
- `transition-timing-function`：动画**缓动**设置，如ease-in、ease-out 等
- `transition-delay`：动画**延迟**设置，填写**秒数**即可


通常搭配`hover`进行使用，
- 定义两个关键帧：**起始帧**和**hover帧**
	- 起始帧和hover帧的**样式不同**，产生transition 的过度![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602170052043.png)
	- 选择变动的类的名字：background
	- 持续的时间：2s
	- 缓动效果类型：ease-in
	- 延迟时常：0s


==过度==的**缓动类型设置**
- 可以定义**标准缓动类型**：ease-in、ease-out 等
- 也可以通过**动效软件**来设置**贝塞尔曲线函数**：cubic-bezier(n,n,n,n)


### 更复杂的动效
除了过度外，CSS 还有一个 **animation 属性**，用来实现**更复杂的动效**
- `animation-name`： 定义动画**事件名**
- `animation-duration`： 动画**完成时间**
- `animation-timing-function`：动画的**缓动效果**
- `animation-iteration-count`：动画的**播放次数**，可是是指定**数字**，也可以是**infinte(无限)**


并且，这个属性要搭配 **@keyframes** 来进行实际设置
- 可以用from to 定义关键帧位置![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602170110021.png)

- 可以用百分比定义关键帧位置![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602170117820.png)


**color**、 **background-color** 等属性可以使用 RGB 和 **RGBA** 色值
- rgb**a**的a是**透明度(alpha)**的意思，实际输入可以这么表示
	- div { color: rbga(255,255,0,0.5); }


**border-radius** 设置**圆角**
- div { border-radius:4px; }


**box-shadow** 设置**阴影**
- div { box-shadow : 4px 4px 5px #000000; }
	- x轴 y轴 模糊值大小 颜色


**overflow** 控制**溢出内容**的**展示状态**，隐藏(hidden)or滚动(scroll)样式
- overflow:hidden![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602170151296.png)

**width** **heigh** 对于长宽的设置，可以使用具体**像素**，也可以用**百分比**同时
**min-width**、**max-width** 可以用来设置尺寸的极限范围区间
