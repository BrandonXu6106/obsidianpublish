## HTML语法
学习 HTML 时， 可以先收藏 [W3C](https://www.w3school.com.cn/tags/index.asp) 的 HTML **标签手册**
HTML编写的基本语法：<起始标签名>+内容+</结束标签名>
- `<h1> 相关内容 </h1>`

不同的标签，会有一些特殊的 HTML 属性
- a标签：可以添加超链接属性和值
	- `<a href ="http://www.baidu.com"> 百度 </a>`

HTML是非常注意**层级结构**的，==标签内可以添加标签==，形成一个上下级的关系，类似图层结构
- 书写方式需要通过”**换行**“和”**缩进**“![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161431421.png)![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161431378.png)

## HTML文档结构
完整的HTML文档，有一个比较正式的**代码结构**![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161434536.png)
1. `<!DOCTYPE html>`： 标准==网页文档声明==
2. `<htmI>`标签：HTML元素顶级标签，表示==文档中的 HTML 部分的开始==
	- `<html>` 与 `</html>` 标签限定了**文档的开始点和结束点**，在它们之间是文档的头部 `<head>` 和主体 `<body> `
	- `<html>` 标签还支持 [HTML 中的全局属性](https://www.w3school.com.cn/tags/html_ref_standardattributes.asp "HTML 全局属性")。
	- 应该始终在 `<html>` 标签内包含 **lang 属性**，以**声明网页的语言**。这旨在帮助搜索引擎和浏览器。
		- `<html lang="zh"> ...</html>
3. `<head>`：所有头部元素的==容器==，容纳==各种元素==
	- 每个 HTML 文档都应该包含一个 head 元素
	- 包含的元素：
		- `<title>` **(必需元素)**
		- `<style>`
		- `<base>`
		- `<link>`
		- `<meta>`
		- `<script>`
		- `<noscript>`
4. `<meta>` 标签：==描述文档元信息、HTML的全局属性、描述文档关键字==
	- 指定名/值元数据对：
		- 为搜索引擎定义关键字：`<meta name="keywords" content="HTML, CSS, JavaScript">`
		- 定义有关网页的描述：`<meta name="description" content="Free Web tutorials for HTML and CSS">`
		- 定义页面的作者：`<meta name="author" content="John Doe">`
		- 设置视口，改善网站在各种设备上的外观：`<meta name="viewport" content="width=device-width, initial-scale=1.0">`
	- 声明字符编码：`<meta charset="UTF-8">` 
	- 模拟 HTTP 标头字段：`<meta http-equiv="refresh" content="30">`--http-equiv和content 属性--每30秒刷新一次文档
	- `<title>`：用来标识页面名称，定义文档的标题![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161532230.png)
5. `<body>`：==包含页面主要内容==，也是个容器，下面包含`<div>` 、`<p>`等
6. `<div>` ：
	- 是一个**块级元素**，它的内容自动地开始一个新行。
	- 可以把**文档分割**为独立的、不同的部分（分区或小节，division/section)。
	- 任何类型的内容都可以放在 `<div>` 标签内！
		- 通过使用 `class` 或 `id` 属性，可以轻松地对 `<div>` 标签设置样式。![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161529807.png)
	- 默认情况下，浏览器总是在 `<div>` 元素前后放置一个换行符。
## HTML文本标签类型
常见的==文本定义==标签类型：
- `<h1>-<h6>`：1级**标题**-6级标题
- `<p>`：表示一个**正文**的段落
- `<b>`：仅**仅加粗**文本字重 (==仅视觉==)
- `<i>`：仅**仅**定义文字**斜体** (==仅视觉==)
- `<strong>`：**加重**文字**权重和凸显**，同时**加粗** (==视觉+语义==)
- `<em>`：**定义**需要**注意**的文字，**同时倾斜** (==视觉+语义==)
- `<del>`：定义**删除线**文字
- `<br/>`：**换行符**，类似word中的换行效果

==列表==的标签类型：
- `<ol>`：**有序**列表
- `<ui>`：**无序**列表
- `<li>`：具体列表**条目**![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161547818.png)

==链接/多媒体==标签
- `<a>`：需要添加 **href 属性**来设置**指向地址** `<a href="http://www.supermancall.com">超人的电话亭</a>`
- `<img>`：**图像**标签，用来显示位图的标签
	- 使用最频繁，比较特殊，**没有结束标签**。包含 src、width、height 等常用标签
	- `<img src="img/logo.png" width="100" height="100">`
- `<audio>`：**音频**标签，可以嵌入一些类似mp3的音频文件进行播放
- `<video>`：**视频**标签，可以嵌入类似avi、wmv、mp4等主流格式的视频

==表单==标签
用来**收集**用户信息的**填表清单**，由**form**标签进行**声明**，表单内具体的**收集信息类型**则由**其它标签**完成：![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161716220.png)
- `<input>`： 一个标准的输入元素
	- **没有结束标签**，主要属性是 **type**，用来定义input 的具体**输入类型**
	- `< input type="email"`>：指定邮箱输入框
	- `< input type="password"`>: 指定密码输入框
	- `< input type="tel">`： 指定手机号输入框
	- `< input type="date">`: 指定日期选择空间
	- `< input type="color">`: 制定拾色器
- `<textarea>`：一个**多行文本**输入框
- `<select>`：**下拉**选项列表
- `<button>`：表单执行的**按钮**

==布局结构==标签
 >专门用来进行内容布局和定义结构的标签有
 
- `<div>`：万能的**模块定义**的标签 (语义划分，无视觉意义)
	- 专门用来进行**聚合内容**的万能标签，类似**UI 设计**软件中的 **“编组“功能**，
	- 多数**网页**，都是由**众多的div标签搭建**对应结构
- `<span>`：行内内联元素
- `<header>`：页面的**头部**内容区域定义
- `<nav>`：页面的**导航**栏区域定义
- `<article>`：页面的**正文**内容区域定义
- `<aside>`：页面的**侧边**栏内容区域定义
- `<footer>`：页面的**页脚**内容区域定义
