## css样式的语法
CSS和HTML的编写有一定的区别每一段完整的CSS，包含两个部分一一 **选择器 + 声明**
- **选择器**：选择要==作用==效果的==对象==
- **声明**：具体针对某个属性的==效果定义==
- 声明中，每个**属性**，由 **“；” 分隔**，最后一个属性末尾也需要加; ![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161736099.png)
- 一条声明中，**一个属性**可设置**多个参数**![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161742668.png)

## css的引用方法 
引用方法：写在**什么地方**，如何产生**作用**
- **外部**样式表
- **内部**样式表
- **内联**样式


### 外部样式表
在这个 HTML 文档之外，**创建**一个 **CSS 文件夹**，**专门**用来**存放 CSS 代码文档**，类似于figma中的外部组件库
- 大大提高了**开发效率**
	- 多个页面，可以共同调用一个css文档


### 内部样式表
**页面内**添加 **style 标签**，来开辟一个专门写 CSS 的地方，以在`head` 中为例：
- 单独控制**一个页面**：![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161751383.png)


### 内联样式
也叫**行内样式**，将样式**直接写进HTML标签**中的做法，用来比较**针对性**的**控制某个标签**的样式
- **不推荐**使用，阅读/书写体验不好
- 属性无法换行，只能写在同一行![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161754371.png)

### 引用选择
可以单独使用，也可以共同使用
样式发生作用的优先级：
- 内联样式>内部样式表>外部样式表


## 选择器的具体认识
常见：
- 标签选择器
- Class、Id 选择器
- 伪类、伪元素


### 标签选择器
- **作用**：直接打在==某一类 HTML 标签上== (同一标签，全局性样式)![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161835361.png)

### Class选择器
**class “类选择器”**：==复数==选择器，可以==作用在多个标签==上 (样式的封装)
- 为标签添加一个**标识**，更灵活的选择内容
- 封装好**一个样式**( ==.class== )，随时在**不同标签**中使用![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161830677.png)
- 同**一个标签**中，可以选择**多个class样式**
	- `<h1 class="simple1 simple2 simple3" >标签1</h1>`
	- simple1 讲字体，simple2 是尺寸，simple3 是下划线，
- class 的**命名规则** :
	- 只能使用**英文/数字**
	- **首字符不能是数字**
	- **不能**和**HTML标签名重复**
	- 除了 **下划线_** 不要使用其他符号
### Id 选择器
**id**：==单数==选择器，==只作用在一个标签==上
- 为标签添加一个**唯一标识**，定义页面中独一无二的元素（如导航栏、页脚）
- 更多使用在 .js中，css**不常用**
- **命名规则**同class

**区别**：
- Class 更适合批量应用样式。
- Id 更适合单个元素的唯一标识。
### 伪类
封装**特殊的类**：选中元素在特定**状态**下的样式。
比如**链接**标签包含**默认**、**选中效果**：![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161858538.png)
- **语法**：`:hover`、`:first-child`、`:nth-of-type(2)`
- **特点**：
    - 不需要额外的 HTML 属性。
    - 常用于交互效果（如鼠标悬停、点击后的状态）。

### 伪元素
用于设置元素**指定部分的样式**。
- 设置元素的**首字母**、**首行**的**样式**
- 在元素的**内容之前**或**之后** **插入内容**
- **语法**：
	- `::before` ：在元素**之前插入**内容。
	- `::after` ：在元素**之后插入**内容。
	- `::first-letter` ：**选择**元素的**首字母**。
	- `::first-line` ：**选择**元素的**首行**。
- **特点**：
    - 常用于**装饰性内容**（如在标题前**加图标**）。

你在做 UI/UX 设计时，可能更关心 **伪类和伪元素的组合使用**，比如按钮的交互效果（`:hover`）和装饰性图标（`::before`）。

## CSS的盒模型
实现元素**排版布局**的核心概念![](https://int-1330323257.cos.ap-guangzhou.myqcloud.com//obsidian/pic/202602161925127.png)
- margin：**外**边距，可以撑开元素和其它元素的距离
- border：**描边**大小，约等于我们设计软件中的描边边框
- padding：**内**边距，从描边到元素制定的空白区域


## 标签尺寸的认识
固定尺寸：wight、height 
- `div {width:100px;height:100px;}`

自适应尺寸：
- width设置为200，height设置成 auto，高会根据内容自动生成
	- `div {width:200px; height:auto;}`
- 内部元素的大小影响标签的实际尺寸，再影响标签的父级尺寸，逐级推导

⭐CSS的宽和高，**只对有”块”化的元素产生作用**，
- 通过添加 ”`display:block;`“ 这段声明，将**标签转化成块**
